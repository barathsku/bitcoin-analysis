# Bitcoin vs. Traditional Assets Analysis

A simple data platform for analyzing the comparative performance of Bitcoin against traditional assets (stocks, fiat currencies).

## Getting Started

Ensure you have Docker installed on your system. Once done, run the following command to start the platform:

```bash
docker compose up -d
```
